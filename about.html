About

This is an online conversion of the 8-Bit Software Magazine, developed by me, Matt Robinson.  8-Bit Software was a public domain software library and floppy disk based magazine for the owners of BBC computers, which was published between 1990 and 1998.  More about 8BS.

Why?

The magazines are available elsewhere online, but not in a format that makes them easy to access or navigate.  Also, searching for obscure BBC information contained in the magazines doesn't usually turn up any useful information from search engines.

Status

All of the magazine issues are now online, but there are still plenty of refinements that I intend to make to the handling of magazine content.  This includes improvements to the Mode 0 conversion and the menus.
