About

This is an online conversion of the 8-Bit Software Magazine using custom software written by me, Matt Robinson.  8-Bit Software was a floppy disk based magazine for the owners of BBC computers, which was published between 1990 and 1998.  More about 8BS.

Background

The magazine disk images are available elsewhere online, but to read the disk images, a BBC emulator (or BBC) is required.  This makes them difficult to access easily, and very difficult to search.  The aim of this site is to present as much of the magazine for viewing online as is practical, as well as enabling users to find obscure BBC information via search engines.

Aims

This project aims to eventually present all 8-Bit Software magazines online, in a standards compliant and accessible format.

Current status

Presently there is only a single issue available as a prototype, while some of the bigger issues are sorted out.  You can find a list of the bugs and enhancements in Bugzilla.
