About

This is an online conversion of the 8-Bit Software Magazine using custom software written by me, Matt Robinson.  8-Bit Software was a public domain software library and floppy disk based magazine for the owners of BBC computers, which was published between 1990 and 1998.  More about 8BS.

Rationale

The magazine disk images are available elsewhere online, but to read the disk images, a BBC emulator (or BBC) is required.  This makes them difficult to access easily, and searching for obscure BBC information contained in the magazines doesn't usually turn up any information from search engines.

Aims

The aim of this site is to eventually present all of the 8-Bit Software magazines for viewing online in as close to their original style as is practical.  This should be in a standards compliant and accessible format.

Current status

Presently not all of the issues are available online, while I develop the conversion and iron out the bugs.  You can find a list of the currently pending bugs and enhancements in Bugzilla.

Comments or suggestions?

If you experience any problems with this conversion, then please raise a bug in Bugzilla for me.  If you'd like to get in touch, or make any suggestions for improvements (technical or otherwise) then please use the contact form.
